urbanX

About urbanX

The inspiration began with me following Gleb Alexandrov Blender tutorial named Game level texturing. My idea was to create a track where you can race on a concrete floor between industrial barrels and pipes.

The track soon got a lot bigger when I added hallways and loading docks with a complete outside area. I later deleted all of this and made a smaller 'technical room', now the room with all the rusty pipes.

I had plans to create more text on the panels, but decided to skip that also, so that's why urbanxa.bmp has lot of empty space.

There are several ways to race this track, so a lot to discover! Beware of the oil!

Software and extra:

  • Modelled in Blender 2.8/2.9
  • Adobe Photoshop
  • Texture are all baked using Blender using a HDRI as a light source.
  • Substance Painter is used to finalize the textures.
  • Worldcut by Jigebren
  • Blender 2.79 Plug-in, created by Marv
  • Blender 2.73 Plug-in, created by Jigebren
    .fob (and custom animation), .taz, .vis placement

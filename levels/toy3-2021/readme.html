About Toy World 3

This track is created for the summer contest 2021 of revoltworld.net.
The track is based on Toy World 2 and I used all the elements from that track to create a whole new world. Lighting is done with vertex baking (sun and some point lights) and with a lot of manual adjustments. Most instance elements are dropped to the ground with help of Blenders rigid body system. I transformed a lot of static objects to custom animation, like the dozer, gator, bear, nelly and robot. I also added several new animations like different toy planes and helicopter. Beware of the planes, they can be dangerous while driving.

Have fun on this track!

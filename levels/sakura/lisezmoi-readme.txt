Petit patch vous permettant de jouer sur Zero Degrees en mode revers.

INSTALLATION : déziper le tout dans le repertoir Re-Volt.
		et remplacer le Temples.inf existant par celui du fichier zip

UTILISATION :	1- avoir le patche 12.07 ou 0916
		2- lancer re-volt et le cheat TRACKER
		3- se placer sur un circuit stock (nhood 1 par ex )
		4- le placer en reverse
		5- se deplacer sur Zero Degrees
		
		ATTENTION IL Y AS UN BUG !!! il vous sera possible de choisir n'importe 
		lequel des circuits xtreme de votre pc et pour la plupart c'est retour 
		windows direct !
		
		6- lancer la course ...
